# Kube Tools

This docker image is based on `docker:stable` and contains the following CLI tools:
- `docker` 18.09.5-ce
- `kubectl` 1.14.0
- `skaffold` 0.32.0
- `kustomize` 2.0.3
- `helm` 2.14.0
- `pulumi` 0.17.20
- `bash` (entrypoint)
- `git`
- `openssh`
- `nodejs`
- `npm`
- `curl`
- `libc6-compat` (for golang binaries)
- `jq`
