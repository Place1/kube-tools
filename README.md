# Kube Tools

This docker image is based on `docker:stable` and contains the following CLI tools:
- `docker` 18.09.5-ce
- `kubectl` 1.15.4
- `skaffold` 0.37.1
- `kustomize` 3.1.0
- `helm` 2.14.3
- `pulumi` 1.0.0
- `bash` (entrypoint)
- `git`
- `openssh`
- `nodejs`
- `npm`
- `curl`
- `libc6-compat` (for golang binaries)
- `jq`
