# Kube Tools

This docker image is based on `docker:stable` and contains the following CLI tools:
- `docker` (docker:stable)
- `kubectl` 1.18.1
- `skaffold` 0.7.0
- `helm` 3.1.2
- `pulumi` 1.14.0
- `vault` 1.2.4
- `kubeval` 0.14.0
- `aws-iam-authenticator` 1.14.6
- `bash` (entrypoint)
- `git`
- `openssh`
- `nodejs`
- `npm`
- `curl`
- `libc6-compat` (for golang binaries)
- `jq`
